Sentiment Analysis.

This is the interface to the Sentiment Analysis Databroker.

To predict the sentiment of an input text, please follow these steps:

  • Click on the button "Predict Sentiment".
  • Enter your query text in the field and click on "Submit Query".
  • Head over to the UI of the Sentiment Analysis Model and "Refresh Results" to see the predictions.
